<h3>GrapheneOS Camera</h3>
<b>This is the new GrapheneOS Camera App based on Android's CameraX Library.
</div><br>
<h2>Features:</h2>
    
- Photography
- Videography
- QR Scan
- Barcode Scan
<br>
<h2>Build:</h2>
Add this in your <i>device.mk</i> file.
</b>

```
PRODUCT_PACKAGES += \
    GrapheneOSCamera
```
